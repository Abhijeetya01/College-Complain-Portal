College Complaint Portal
📌 Project Overview

The College Complaint Portal is a web-based application that allows students to submit and track complaints related to college facilities, academics, or administrative issues.

This system provides an easy and transparent way for students to report problems while enabling administrators to manage and resolve complaints efficiently.

🚀 Features

Student Registration and Login

Submit Complaints Online

Track Complaint Status

Admin Dashboard for Managing Complaints

Complaint Categories (Hostel, Academic, Infrastructure, etc.)

Simple and User-Friendly Interface

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Abhijeetya01/College-Complain-Portal.git
2️⃣ Go to Project Folder
cd College-Complain-Portal
3️⃣ Install Backend Dependencies
cd backend
npm install
4️⃣ Start the Server
node server.js
5️⃣ Open Frontend

Open the frontend folder and run index.html in your browser.
